# How To sshauto

Open multiple ssh+screen terminals (default terminal xfce4-terminal)
```
  Use:	sshauto host1 host2...

   first use)
        no arguments for interactive console to set parameters
   -d | --domain)
         sshauto -d domain host1 host2 host3...
   -h | --help)
      prints this help
   -v | --version)
      prints current version
```
